                                         Odin Admin Dashboard
                                        (This is a static site)
 This Odin project shows how to create layouts using Grid, Flexbox, and all that was taught in Intermediate HTML and CSS.

 Features
 - Navigation links highlighting on hover.
 - Hover effects on heading buttons.
 - Search bar border shows when bar is clicked.
 - Project cards have project ideas with descriptions for intermediate HTML and CSS.

 Avatars are AI images created by me using Midjourney and Ideogram.
